寫法

SASS 有兩種寫法

一種是比較常見的 (.scss) ，另一種是用縮排方式的 (.sass)


一般的 CSS 寫法是這樣，每一個項目都會通通寫出來


(.scss) 的寫法會省略一些重複的東西，但是編譯出來還是會像一般的寫法那樣


(.sass) 的寫法是用縮排的方式，沒有任何的符號


$ 變數

用一個 $ 符號來設定通用的變數，前面是變數名稱，後面是設定值

這樣要管理和修改都會方便很多


import 切分檔案

可以把檔案分成好幾支專門的檔案

像是 _reset.SCSS 就是專門寫 css reset 的檔案

_buttons.SCSS 就是專門寫按鈕的檔案之類的

前面加上 _ 下底線表示不要被編譯成 CSS 檔


要用到那支檔案時，就 import 那個檔案

這樣每個專案都可以直接使用，不用每次都重寫了

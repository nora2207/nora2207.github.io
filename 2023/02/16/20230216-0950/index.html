Flex 基本觀念

外面有容器，裡面是元件

外面綠色包起來的是 Flex 外層屬性 外容器(Container)

裡面咖啡色的部分則是 內元件(items)


外層屬性 Container

  • 如果要讓 div 並排，必須在父層的 Container 下 display: flex;

  • 排版時如果下了 flex ，裡面 item 的 div 會依照寬度的比例自適應伸縮

  • 如果其中一個 div 高度較高，預設是會全部等高 (其他的也更著變高)


Flex 軸線

flex-direction 可以決定 flex 軸線的方向

而軸線又可以分成 主軸 (justify-content) 和 交錯軸 (align-items)


主軸方向 flex-direction

flex-direction 是寫在 Container 裡面

決定 flex 的軸線是怎樣的方向

(一) 從左到右

預設是從左到右，所以不寫的話就是由左到右排列

1 2 3


(二) 從右到左

3 2 1


(三) 從上到下

1
2
3


(四) 從下到上

3
2
1


主軸對齊方式 justify-content

justify-content 決定了主軸的對齊方式


假設以 flex-direction: row(default) 也就是以預設的情形來說

(下面的示意圖中，左右的 | 代表容器)

(一) 從主軸的起點來對齊

|1 2 3 ……|


(二) 從主軸的終點來對齊

|…… 1 2 3|


(三) 置中對齊

左右留白，內容置中對齊

|… 1 2 3 …|


(四) 左右留白分配

左右留白，內容平均分配

|.1..2..3.|

或

每個內容的左右留白都是一樣的

|..1..2..3..|


(五) 左右貼齊，內容間留白

|1…2…3|


交錯軸對齊方式 align-items

align-items 決定了交錯軸的對齊方式

交錯軸會和主軸垂直

這裡也假設以 flex-direction: row(default) 也就是以預設的情形來說

(下面的示意圖中，左右的 | 代表容器)


(一) 靠上排列

從交錯軸的起點來對齊

|1 2 3 ….|
|…………..|
|…………..|


(二) 靠下排列

從交錯軸的終點來對齊

|…………..|
|…………..|
|1 2 3 ….|


(三) 垂直置中

|…………..|
|1 2 3 ….|
|…………..|


(四) 會以滿版的方式來呈現

|1 2 3 ….|
|1 2 3 ….|
|1 2 3 ….|

(三個1是同一個，只是被拉長了，2和3也是一樣)


(五) baseline


換行 flex-wrap

(一) 不換行

預設是不換行，會等比例縮小

|1.2.3.4.5.6|


(二) 換行

超過的部分換換到下一行去

|1.2.3.4|
|5.6……|


(三) 反向換行

從下面開始排列，超過的部分換換到上一行

|5.6……|
|1.2.3.4|

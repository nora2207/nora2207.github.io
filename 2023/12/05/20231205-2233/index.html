innerHTML 增加HTML網頁結構

可以增加 HTML 的網頁結構

組完字串後，用語法渲染到網頁上

原本的東西會被刪除掉，網頁上會顯示新的內容




也可以和 for 搭配使用

這裡用國立自然科學博物館特展JSON資料來練習


ul 裡面沒有寫東西


js 則是先用 querySelector 選取這個 ul

然後算出資料的長度

接著先寫一個空字串，等等組字串時會用到


接下來是跑 for 迴圈，在裡面寫下 li 的內容

有圖片、展覽名稱、地點和展出的期間，依序放到 li 裡面

最重要的是 str += content;

如果沒有這樣寫，innerHTML 每跑一次會蓋掉前面的資料

最後網頁上只會有最後一筆資料而已


最後用 innerHTML 渲染到網頁上就完成了




createElement 增加HTML網頁結構

用 DOM 節點的方式來新增

和 innerHTML 的差別是會保留原本的內容

新增的東西會依序加在後面


HTML 中原本有一個 P段落


用 createElement 增加 HTML 標籤的元素，然後在 JS 中動態增加

接著用 appendChild 把剛剛的節點掛在 .title 中

新增的東西會排在原本的往內容的後面

所以粉紅色的 13579 會在原本網頁內容 123 的後面





也可以和 for 搭配使用

js 則是先用 querySelector 選取這個 ul

然後算出資料的長度


接下來是跑 for 迴圈

先增加 li 這個元素

然後新增展覽的標題放到 li 裡面

接著用 appendChild 在 list 中加入剛剛組好的內容

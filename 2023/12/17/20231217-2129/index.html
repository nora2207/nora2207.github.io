event 物件

可以知道當下的元素資訊

使用 function(e) 就可以了解這個事件的詳細資訊


value

取值

使用 .value 來取出元素，取出來的值為字串

以 input 欄位說明

結果會顯示 value 的內容


賦予值

延續剛剛的範例，也可以改 value 的內容

這樣網頁顯示出來的內容就從 HELLO 變成 今天天氣真好


addeventListener 事件監聽

可以監聽網頁 Event 的事件

裡面的項目分別是 addEventListener(事件,函式,布林值)

第一個是選取的事件，像 click 就是點擊時

中間是匿名 function，也就是要觸發的內容

最後一個是布林值，決定要不要產生冒泡事件

false 為冒泡事件 (Event Bubbling)，會從指定元素往外找

當點擊最近元素時，會由最近的元素來觸發事件

true 為捕獲事件 (Event Capturing)，會從最外層找到指定元素

如果不寫的話，預設是 false


click 點擊事件

也就是當滑鼠點擊時，會觸發效果

像下面的範例那樣

當點擊那個按鈕時，就會跳出 你剛剛點擊了


html 先寫一個按鈕

用 querySelector 選取剛剛的按鈕

然後用 addEventListener 來監聽

當點擊按鈕發生 click 事件時，，就會觸發函式的內容




剛剛的 JS 也可以直接放入一個函式

keydown 鍵盤事件

如果要監聽使用者 keydown 鍵盤事件

把剛剛的 click 改成 keydown

這樣當按鍵盤時就會執行函式內容了


keyCode 鍵盤代碼

而 keyCode 則是按下鍵盤時的鍵盤代碼

像空白鍵是 32，enter 則是 13 之類的

這樣就可以設計當按到什麼鍵盤時會產生什麼效果


blur 離開焦點時觸發

像是點擊 input 欄位時，就會聚焦在那個欄位

如果沒填寫內容，使用 blur ，在離開這個焦點時就可以觸發事件

像是跳出沒填寫內容的提醒之類的


mouse 滑鼠滑入觸發

使用 mousemove 就可以在滑鼠滑入指定內容時觸發事件




stopPropagation 中止冒泡事件

在 addEventListener 的 function(e){} 中加入 e.stopPropagation();

就可以中止不要再去找其他的監聽事件

也就是說如果有重疊的行為，只做自己的事件


preventDefault 取消預設觸發行為

e.preventDefault(); 可以取消元素的默認行為

原本點連結會跳轉到指定的網頁，用了這個之後，可以取消觸發行為


target 和 nodeName 了解目前所在元素位置

使用 e.target 可以知道目前滑鼠點的地方是哪個 DOM 物件

而 nodeName 可以知道目前點選的節點名稱


當點擊 a 連結時，console.log 會顯示

當點擊 li 連結時，console.log 會顯示

這樣要設計東西時，就可以清楚知道要點到什麼才會產生效果


我用不同顏色的 border 來包住，這樣看比較清楚

紅色框框是 ul，藍色框框是 li，黑色框框是 a 連結




網頁座標

screen 是螢幕解析度的大小

page 是整個網頁內容的頁面大小

client 是目前開啟的瀏覽器視窗的大小

製作 banner 區的圖片輪播

這裡要來製作一般網頁常見的 banner 圖片輪播

我試了二種方法，一種是參考別人單純使用 CSS 及 HTML 的方法

另一種是用 Swiper 套件來製作，弄好久終於成功了


首先，圖片部分設定了二組，各五張

一組是桌面版的圖片，場景比較廣、比較遠

另一組是平版手機版的圖片，畫面有拉近的感覺

斷點則是設定在 767px 以下時會是平板手機版的圖片， 768px 以上時是桌面版的圖片


max-width，表示這個數字以下（包含） 的都適用。(<=)
min-width，表示這個數字以上（包含） 的都適用。(>=)


因此用 max-width: 767px (當畫面小於 767px時)

desktop-only (桌面版的圖片) 會 display: none (隱藏起來)

只有顯示平板手機版的圖片

而當 min-width: 768px (當畫面大於 768px時)

mobile-only (平板手機版的圖片) 會 display: none (隱藏起來)

只有顯示桌面版的圖片



另外，圖片也有加入

變成響應式圖片

圖片會隨著視窗的大小來調整比例

下面也不會有卷軸出現


淡入淡出圖片輪播

這一種是淡入淡出的形式來切換圖片，只有用到 html 及 CSS



有5張圖片，整個週期是20秒

所以每4秒會淡出淡入的換下一張圖片

當滑鼠停在圖片上時，會停止切換圖片，停在當下的畫面


使用 Swiper 套件

如果只有一組桌面版的圖片，直接照著官方的說明套用就可以了

但我還多了一組平板手機版的圖片

本來直接加在桌面版的圖片下面，結果下面的分頁點點變成10個

輪播時變成一張有圖片，一張空白的方式在輪播

後來把2組圖片各自獨立用 div 包起來

但是只有前面的那組成功，後面的那組會停在第一張圖片，左右切換也沒辦法用

最後乾脆直接把最外面的 div (swiper) 設定了2組，結果就成功了！



左右的箭頭有改成有點透明的灰色，滑鼠移過去會變藍色


接著來說一下 JS 的設定

cssMode: true 預設是 false ， true 表示套用 cssMode 這個樣式

loop: true 預設是 false ， true 表示圖片會循環，第五張之後會回到第一張這樣循環

clickable: true 可以直接點選下面分頁的點點來切換圖片

autoplay: { delay: 3000,} 自動輪播圖片， 3000 表示3秒換一張圖片


另外在補充比較常用的設定

direction 輪播方向，預設是 horizontal 水平的，也可以改成 vertical 垂直的

effect 輪播效果，預設是 'slide'，其他還有 'fade' 'cube' 'coverflow' 'flip' 'creative' 可以選擇

滑鼠移過去才會出現的特效
Nora
  2023-05-06 20:05:02 2023-05-06 20:05   2023-10-27 10:51:46  
  • CSS
  
  • 滑鼠特效
  • hover

如果想要保持畫面的乾淨美觀，但又想要有說明文字

就可以用這個方法

一開始先隱藏起來，滑鼠移過去時才會出現




先設計一個圖片列表，然後下面有說明文字


在 li 設定 position: relative 和 overflow: hidden


裡面的說明文字設定 position: absolute ，接著把位置設定在 bottom = 負值 ，說明文字就會因為超出 li 的範圍而隱藏起來


這時只要設定一個 hover 時 bottom = 0 ，說明文字就會出現了


然後因為不要出現的那麼突然，所以有加一個漸變 transition: all 1s ，讓說明文字可以慢慢出現和消失

用 jQuery 來製作下拉式收闔選單
Nora
  2023-04-18 13:49:44 2023-04-18 13:49   2023-10-30 18:54:53  
  • jQuery
  
  • 下拉式選單
  • dropdown
  

一般網頁常見的下拉式收闔選單

有一種是點了會展開選單，再點會收闔起來

另一種是滑鼠移過去就會展開，移走就收闔起來

這裡會使用 jQuery 來製作第一種點擊的版本


點產品專區會有輕食、甜點、咖啡的選單出現

再點一次產品專區會收闔起來



HTML 部分

首先在產品專區的這個 li 中再加入一組輕食、甜點、咖啡的 ul 和 li

在這組 ul 設定 dropdown-open 的 class

接著在產品專區的 a 連結中設定一個叫 dropdown 的 class



然後是 CSS 部分

設定首頁、產品專區、常見問題的 hover 和 active 樣式


然後要把輕食、甜點、咖啡的 ul 先隱藏起來


最後是 jQuery 的部分

先用 preventDefault 讓連結不會連到其他地方

然後用 toggleClass 增加產品專區的 active 的樣式

最後是用 slideToggle 來使下拉選單下滑/上滑

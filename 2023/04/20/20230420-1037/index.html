表單結構

表單的最外層用一個 <form> 包起來

裡面的輸入欄位則視需要來設計需要的項目

最後則使用是 submit 來傳送資料

類似像這樣



submit 是用 <input> 標籤，在 type 寫上 submit

value 寫上要顯示的文字


placeholder 和 label

如果輸入欄位不要空白，可以在 placeholder 填上內容

這個只會提示在畫面上，要輸入內容時就會不見了

就像是範例的 請輸入電子郵件 這幾個字


如果輸入欄前面想要有項目名稱，可以在輸入欄位前加上 <label>

另外，以這個範例來說，如果想要點電子郵件的文字，就會直接跳到對應的輸入框的話

可以在 <label> 裡面填上 for ，和它對應的 <input> 裡面則填上 id

記得這2個填的內容要一樣



radio 和 checkbox

如果是單選，就使用 radio ，如果是複選，就使用 checkbox

radio 的寫法像下面這樣

在 <input> 的 type 寫上 radio

name 的部分則是這一組的 radio 都要寫一樣，這樣才能單選

value 則是送出時，後端資料庫會收到的資料

(只會看到 value 內容，並不會看到網頁上表單的其他內容)

最後面則填上要顯示在畫面上的名稱


checkbox 部分像是這樣

在 <input> 的 type 寫上 checkbox

name 的部分也是這一組的 checkbox 都寫一樣

value 一樣也是送出時，資料庫會收到的資料

最後面也是填上要顯示在畫面上的名稱



select

select 是下拉式選單

首先，新增一個 <select> 標籤， name 填入這個選單的內容

選項則是用 <option> ，要幾個選項就填幾個 <option>

value 一樣也是送出時，資料庫會收到的資料

像這個就會收到 age = 3 的資料

所以 name 不能隨便取，不然會很難看懂到底填了什麼



textarea

<textarea> 是多行文字輸入，可以輸入較多的內容

cols 是每行可以輸入多少字元

rows 是總共可以輸入幾行

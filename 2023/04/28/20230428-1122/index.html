圓弧效果

可以利用 CSS 去設計一些圓弧的效果



主要是利用 border-radius 來控制

像這裡的

後面四個數值分別代表 左上 右上 右下 左下

改變數值就可以改變形狀


要在這個形狀的外面加上邊框也是可以的

就用之前說過的 border 就可以在外圍加上邊框


另外，如果裡面有打文字

想要水平置中就使用 text-align: center

如果只有單行，想要垂直置中時

可以看這個容器的高度是多少， line-height 就設定一樣

背景漸層效果



由上到下

只要加上 background: linear-gradient(上面顏色,下面顏色)

這樣就可以做出漸層的效果

這個漸層是由上到下，這個是預設的


由左到右

如果要變成由左到右

就在前面加上 to right

就會從左邊的紫色漸層到右邊的橘色了


由右到左

要改成由右到左也是和由左到右差不多

只是變成 to left

就會從右邊的紫色漸層到左邊的橘色了


由左上到右下

要改成左上到右下的漸層方式，就用 to bottom right

這樣就會從左上的紫色漸層到右下的橘色


由右上到左下

同理，如果是要由右上到左下，就改成 to bottom right

這樣就會從右上的紫色漸層到左下的橘色


多顏色漸層

如果想要多個顏色漸層，就在後面加入多個顏色

就會從第一個顏色漸層到最後一個顏色了


文字漸層效果

CSS 也可以將文字顏色改成漸層的效果



先將文字的顏色改成透明的 color: transparent

接著是設定背景顏色

background-image: linear-gradient(purple, orange)

預設是由上到下的漸層方式，也可以利用剛剛背景漸層的方式來改變漸層方向

最後是將背景裁切成文字範圍 background-clip: text 就成功了


陰影效果

使用 box-shadow 來增加陰影效果



後面四個分別是 box-shadow: X軸 Y軸 擴散值 顏色


半透明背景

前面是顏色，最後一個是透明度，數字越小越透明

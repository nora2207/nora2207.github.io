使用 CSS 來製作下拉式收闔選單
Nora
  2023-04-21 09:55:22 2023-04-21 09:55   2023-10-27 16:48:39  
  • CSS
  
  • 下拉式選單

一般網頁常見的下拉式收闔選單

有一種是點了會展開選單，再點會收闔起來

另一種是滑鼠移過去就會展開，移走就收闔起來

這裡會使用 CSS 來製作第二種滑鼠移過去的版本


二層選單

滑鼠移到關於我們和產品專區時，會有下拉選單出現



主要是先把下拉選單隱藏起來，把 ul 先 display: none

等到滑鼠 hover 時再出現

滑鼠移到 menu 裡的 li 時，就把 ul 打開顯示出來


三層選單

滑鼠移到關於我們時，會有歷史發展和得獎紀錄

滑鼠再移到得獎紀錄時，又會再有年份的下拉選單



做法和前面的差不多

比較不一樣的地方是滑鼠移到 menu 的 li 時，就把裡面的 ul 打開顯示出來

年份的這層選單也要設定一下位置，才不會接在原本選單下面出現

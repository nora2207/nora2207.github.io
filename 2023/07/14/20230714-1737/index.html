變數(Variable)和值(Value)

像是這裡寫

a 就是變數，而 1 就是值


基本型別(primitives types)

如果要查型別，可以用 typeof (後面放上要查的東西)

像是 typeof 1 就會出現 ‘number’


數字 (Number) 型別

就是數字，像是 1 2 3 之類的

數字可以拿來計算

  • 如果沒有()就是先乘除後加減

  • Number 型別可以為負數

  • Number 型別可以有小數點

常用的方法有下面這些

+ 加 2+3=5

- 減 3-1=2

* 乘 2*3=6

/ 除 6/2=3

% 取餘數 20%3=2

** 次方 22=2*2=4 23=222=8


練習一下變數和數字：
小明去買蘋果，小明的錢包有200元，蘋果一顆30元，他買了6顆，付完錢還剩下多少錢？

let wallet = 200;
let applePrice = 30;
let appleNum = 6;
let total = wallet - applePrice * appleNum;
最後輸入 total 就會出現剩下 20 元了


字串 (String) 型別

用 “” 或 ‘’ 包起來

像是 “你好”


布林 (Boolean) 型別

true false


未定義 (undefined)

有變數名稱，但是還沒被賦予任何值

預設就會是 undefined


空值 (null)


Symbol

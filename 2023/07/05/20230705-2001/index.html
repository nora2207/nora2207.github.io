lightbox 是一個很方便的 jQuery 套件

分成單張和多張的方式

單張就是每張圖片要分開點

看完一張要看下一張時要先關掉這張圖片再點下一張

而多張是點了一張圖片後，會有左右的箭頭可以直接點到下一張或前一張圖片

做出來的效果像這樣 ↓




首先是要載入檔案

CSS 要放在 <head> 最前面

然後 Javascript 要放在 </body> 結尾前面


然後是 html 部分

單張和多張主要就是差在 data-lightbox 的設定

這個名稱可以自訂，同一個名稱就會是一個圖片群組，就可以左右切換

上面的 data-lightbox 一個是example-1，一個是example-2，就會是單張的形式

如果兩個都設定成example-1，就變成多張可切換了


CSS 則單純是改成自己要的樣式

像我有把圖片改成圓角


jQuery 的部分在lightbox 網站上有說明

這裡挑幾個來介紹一下


fadeDuration 開啟關閉圖片時的淡入淡出效果，預設是 600 就是 0.6 秒

resizeDuration 是圖片轉場的秒數，預設是 700 就是 0.7 秒

wrapAround 預設是 false 圖片不循環， true 是圖片可以循環(最後一張可以再點到第一張)

showImageNumberLabel 是群組圖片下方的說明，預設是 true

albumLabel 群組圖片下方的說明，預設是 "Image %1 of %2"

漢堡選單就是在行動版時會有三條線長得很像漢堡的選單

像是這樣 ↓




html 除了原本的選單外，再加入一個漢堡圖示的 a 連結

這裡的圖示是用 Font Awesome

在 html 中是加入了三條線的圖示

但是一開始先隱藏起來(行動版時才會出現)


CSS 部分則是先設計PC版的樣式

用 float 讓選單變成靠右橫排


接著設定行動版的樣式




jQuery 的部分分成兩個，這兩個都是行動版時會用到的

第一個是點了漢堡圖示後，會切換 menu 的高度變 400px 或 0px

這樣就可以打開或關起選單


這一個是點了漢堡圖示後，會切換成叉叉圖示

toggleClass 裡面放的是 Font Awesome 的叉叉圖示 class 名稱

這樣就可以切換漢堡和叉叉這兩個圖示了

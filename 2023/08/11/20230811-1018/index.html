用 parallax.js 來製作視差滾動特效
Nora
  2023-08-11 10:18:00   2024-08-31 19:26:14  
  • jQuery
  • jQuery
  • parallax
  • Scrolling
  

視差效果是利用各種元素間不同的移動速度造成的

使用 parallax.js 就可以很簡單的做出視差滾動的特效

像是下面這個，滑鼠在滑動時，會覺得背景圖也跟著一起動




首先要載入 jQuery 和 parallax.min.js

然後在 HTML 中在要製作特效的地方加入 data-parallax="scroll"

圖片則使用 data-image-src="圖片網址" 來加入圖片

圖片顯示的位置預設是 center center 要改變可以用 data-position="top" 或其他方向位置

data-bleed="10" 可以讓畫面不要有卡卡的感覺，預設是 0

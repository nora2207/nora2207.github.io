控制判斷 - if、else if、switch

if

if () 是判斷式

eat(); 是陳述式

如果判斷式()裡面的內容成立，就會跑{}裡面的內容

所以這裡就是如果肚子餓，就會執行 eat(); 函式


範例-滿20歲可投票

輸入出生年份會計算是否已成年可參與投票

如果滿20歲會顯示可投票

未滿20歲，則會提示還有幾年才能投票


設計輸入框可以輸入出生年份


計算是否滿20歲

年齡 = 今年 - 輸入的年份

new Date() 可以取得現在的日期時間
getFullYear() 可以取得現在的年份


使用 if 和 else

如果年齡大於或等於20歲

會顯示 “滿20歲可以投票囉！”

其他情形則顯示

“未滿20歲不能投票唷，還要再等” + num +”年才能投票唷”

num 為用20減剛剛算出的年齡來判斷還要幾年才20歲


(點 Result 會跳出訊息視窗)



else if

假設飢餓程度為 1一點點餓 ~ 10超級餓

如果飢餓程度大於等於7，那就執行 eat('便當');

前面沒有執行就會繼續跑後面的條件

如果飢餓程度在4~7之間，那就執行 eat('沙拉');


如果有多個條件可以多寫 else if

最後可以寫 else ，如果所有條件都沒有符合，最後一定還是會跑 else 的內容


範例-及格與補考

輸入成績，進行變數的存取與判斷

來通知是否及格與補考


設計輸入框輸入成績


85分以上為”優秀”

60分以上為”及格”

50分以上，60分以下為”可以補考”

50分以下為”不及格”


(點 Result 會跳出訊息視窗)



switch

switch () 是表達式，裡面包含條件

case: 比對資料，如果符合就執行程式碼

break; 放在 case: 後方，讓程式碼不要繼續執行

default: 類似 else 的功能，前面都不符合就執行這一個


範例-今天星期幾

使用內建函式來產生目前系統是星期幾


new Date() 可以取得現在的日期時間
getDay() 可以取得現在的星期


取得當下的星期


使用 switch 來判斷

當結果是1時，會跳出 “今天是星期一”

2 ~ 6 以此類推

最後的 default 則是以上都不是時

會跳出 “今天是星期日”


(點 Result 會跳出訊息視窗)



範例-保留訂位

用 confirm() 來詢問是否要保留訂位

可以選擇確定或取消

如果選擇了確定

就是 case true，會提示保留訂位

選擇取消就是 default 的取消訂位


(點 Result 會跳出訊息視窗)



if 和 switch 的差別

if 執行時會全部跑一次，看符合那一個條件再去執行

switch 則是先看哪個條件符合，直接執行符合的那一個

所以 switch 的效能會比較好，但是條件可能也要帶很多個進去設定

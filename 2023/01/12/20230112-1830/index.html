  • <blockquote> 定義長的引用

清單

  • <ul> 沒有順序的清單 (預設黑點點)

  • <ol> 有順序的清單 (1. 2. 3.)

  • <li> 清單內的項目

  • <dl> 描述性清單(最外層)

  • <dt> 描述性清單項目

  • <dd> 描述性清單項目的描述 (內容會稍微內縮)


下拉式選單

  • <select> 下拉式選單

  • <option> 下拉式選單中的選項，搭配 value="值"

  • <optgroup> 下拉式選單群組，搭配 label="看到的群組文字"

  • <datalist> 可以輸入也可以直接選的下拉式選單

搭配 <input> 可以篩選文字選項，input 中的 list 的值等於 datalist 中的 id


有標示 selectd 的選項是網頁中會看到的預設選項
size 為網頁上可以看到的選項數目
multiple 為可以複選的下拉式選單


圖像

  • <img> 圖片

搭配 src="" 和 alt="圖片有問題會出現的字"


  • <figure> 獨立的圖片或圖表


  • <figcaption> 定義 <figure> 的標題


  • <map> 可點選區域的圖像


  • <area> 可點選區域的小區塊

搭配 coords="座標"

框架

  • <iframe> 設定頁面框架，搭配 src="網頁"

網頁版面

  • <div> 區域

  • <section> 頁面中的區段

  • <nav> 常用在導覽列

  • <aside> 常用在側欄

  • <header> 頁首，放置主要內容

  • <footer> 頁尾，放置資訊(作者、電話、地址…)

  • <article> 文章

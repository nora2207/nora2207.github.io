基本介紹

border 的語法分別是 線條粗細 線條樣式 顏色

像這個範例就是 1px 的黑色實心線


線條位置

border 線條的順序是上 右 下 左

除了使用 border 一起設定外，也可以單獨設定

border-top // 上面的線

border-right // 右邊的線

border-bottom // 下面的線

border-left // 左邊的線


線條樣式

比較常用的有下面這幾個：

(這裡的線條粗細是用 5px 看比較清楚)

solid // 實心線

實心線

dashed // 虛線

實心線

dotted // 圓點虛線

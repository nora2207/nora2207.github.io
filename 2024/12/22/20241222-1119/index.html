Bootstrap 欄的運用
Nora
  2024-12-22 11:19:00   2025-01-05 11:24:39  
  • Bootstrap
  

格線基本架構

最外層是 container，接下來是 row (橫列)，再來是 col (欄)

一般是分成 12 col，所以 col-6 就是左右2欄




外層至少要有一層 container，不然會出現水平卷軸


寬度運用

限制單欄寬度

預設為 12 欄，要設定為3大欄，可以用 col-4




換行

欄線如果超過12會自動換行




等寬欄

col 沒有加上數字時，會自動平均分配

例一：3個都沒有數字，3個平均分配

例二：第一個是 col-6 ，後面2個平均分配剩餘空間




彈性寬度

加入 col-auto，寬度會依照內容來調整




巢狀

外層有一個格線結構，裡面再加入另一個



響應式

範例一

要求：

  1. 桌面版 xl：4 大欄
  2. 平板 md：3 大欄
  3. 行動版 xs：2 欄

製作順序：1行動版 2平板 3桌面版

class 加上 col-6、col-md-4、col-xl-3



範例二

要求：

  1. 平板以上為 md：3 大欄
  2. 行動版 xs：單 欄

製作順序：1行動版 2平板以上

class 加上 col-md-4

(當有其他 col 中斷點時，可不加入 col-12)



推移 offset

目前大多會透過 flex 特性來推移

offset 的運用情境比較少

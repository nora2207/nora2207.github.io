Bootstrap 的文字設定
Nora
  2024-12-01 20:59:00   2025-02-11 17:35:08  
  • Bootstrap
  

標題標籤

直接使用 h1 ~ h6 的標籤




class 名稱

不能直接使用 HTML 元素時，可以使用 .h1 到 .h6 的 class 名稱




small 標籤

使用 small 標籤的字會比較小一點點




顯示標題

如果覺得預設的標題不夠大

可以在 class 加上 display 1 到 6 ，這樣字體會放大很多




文字大小

使用 fs-{ 1-6} 來調整文字大小

只有單純切換文字的大小，沒有額外的 margin 或 padding



字體變化

font-style 通用類別的縮寫為 .fst-*

font-weight 通用類別的縮寫為 .fw-*

不是所有的字體都可以套用，有些字體會沒有變化




大小寫轉換

  • text-lowercase 全部小寫

  • text-uppercase 全部大寫

  • text-capitalize 首字大寫

行高

通常預設是1.5，如果沒有特別需要可以不用調整



等寬字體

用 font-monospace 可以把字體變等寬



Reset color

用 .text-reset 來重置一個文字或連結色彩，以便它們繼承父層的顏色

下面的例子原本連結是藍色，加入 text-reset 後，會套用父層的灰色




文字裝飾

加入 .text-decoration-{} 來改變文字裝飾

  • underline (下底線)

  • line-through (刪除線)

  • none (沒有線)




前導主題

在標題之下用來介紹用的敘述，想要顯眼一點

但是並不是標題，就可以用前導主題

使用方法是在 class 加上 lead




行內文本元素

加上各種標籤，就會有不一樣的效果




縮寫

加入 abbr 標籤




引用

使用 blockquote 標籤

如果 class 用 blockquote，字會比較大一些



引用來源

用 figure 標籤包住




文字對齊

置中對齊是在 class 的地方用 text-center

靠左對齊是用 text-start，靠右對齊是用 text-end




如果桌面版要靠左，行動版要置中，可以使用 text-center text-lg-start



垂直對齊

搭配 inline 元素

inline 中不太建議調整 align-*，這會受到字體不同影響其結果




搭配 table cell 元素

用在外層表格或單一儲存格都可以

實戰中常用技巧，尤其是 align-middle




換行

用 .text-wrap 可以把文字斷行


用 .text-nowrap 則可以防止換行


用 .text-break 設定超出容器部分換行，預防長文破壞排版



文字截斷 Text truncation

較長的內容，可以加上 .text-truncate，會變成刪節號截斷文字

(需為 display: inline-block 或是 display: block)



標題使用 text-truncate，超出範圍會顯示 ...

內文部分的單行高度為 1.5rem，所以可以用 style="height: 4.5rem" 限制高度只顯示3行

第4行之後的部分用 overflow-hidden 隱藏掉了



列表

無樣式

在 class 加上 list-unstyled 清除項目點的樣式




行內

ul 的 class 加上 list-inline

li 的 class 加上 list-inline-item

這樣就可以三項 li 排成一排，而且有一點點間距




描述型列表對齊

使用格線系統進行排版

外面大標籤是 dl，裡面標題是 dt，裡面內文是 dd



這裡面最後一項在 class 加上 text-truncate

文字太長時會自動截斷變成…

Bootstrap 的文字設定
Nora
  2024-12-01 20:59:00   2024-12-17 23:12:14  
  • Bootstrap
  

標題標籤

直接使用 h1 ~ h6 的標籤




class 名稱

不能直接使用 HTML 元素時，可以使用 .h1 到 .h6 的 class 名稱




small 標籤

使用 small 標籤的字會比較小一點點




顯示標題

如果覺得預設的標題不夠大

可以在 class 加上 display 1 到 6 ，這樣字體會放大很多




前導主題

在標題之下用來介紹用的敘述，想要顯眼一點

但是並不是標題，就可以用前導主題

使用方法是在 class 加上 lead




行內文本元素

加上各種標籤，就會有不一樣的效果




縮寫

加入 abbr 標籤




引用

使用 blockquote 標籤

如果 class 用 blockquote，字會比較大一些



引用來源

用 figure 標籤包住




對齊

置中對齊是在 class 的地方用 text-center

靠左對齊是用 text-start，靠右對齊是用 text-end




列表

無樣式

在 class 加上 list-unstyled 清除項目點的樣式




行內

ul 的 class 加上 list-inline

li 的 class 加上 list-inline-item

這樣就可以三項 li 排成一排，而且有一點點間距




描述型列表對齊

使用格線系統進行排版

外面大標籤是 dl，裡面標題是 dt，裡面內文是 dd



這裡面最後一項在 class 加上 text-truncate

文字太長時會自動截斷變成…

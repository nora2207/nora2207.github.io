Bootstrap 的表格設定
Nora
  2024-12-12 20:39:00   2025-01-05 11:02:40  
  • Bootstrap
  

基本表格

和一般的表格一樣

最外層有個 table 標籤，裡面分成 thead、tbody、tfoot

<table> 的 class 加入 table 會自動套用表格樣式

<thead> 是表格的表頭，<tbody> 是表格裡面的內容，tfoot 是表尾

表格裡面的內容是由 tr、th、td 組成

<thead> 和 <tbody> 中都有 <tr> ，<tr> 裡面再有好幾個 <th> 的同一行資料



colspan="2" 可以一次跨2欄


變數

使用 class 把表格列或是單元格上色(表格的顏色會比較淺)

像是 class="table-primary"



預設的顏色： primary 淺藍 secondary 淺灰 success 淺綠 info 淺亮藍 warning 淺黃 danger 淺紅 light 偏白的灰 dark 黑

表格強調

在 <tbody> 範圍裡的 class 加上 table-striped

將表格行增加條紋的樣式




可滑入行

在 <tbody> 範圍裡的 class 加上 table-hover

滑鼠移過去時會變色，類似 hover 的效果




突顯表格

class 加上 table-active 可以凸顯表格行或是單一格




表格邊框

有邊框

class 加上 table-bordered 在表格和儲存格加上邊框




無邊框

在 class 加上 table-borderless 可以讓表格沒有邊框




邊框顏色

在 class 加上 border-顏色變數 可以改變邊框的顏色

像下面這個就是加上 border-info




表格縮小

在 class 加上 table-sm 可以把儲存格的 padding 縮減一半，讓表格更精簡


(上面是縮小的表格，下面是原本的)



垂直對齊

在 class 加上 align-middle 可以置中對齊

align-bottom 向下對齊，align-top 向上對齊




巢狀表格

巢狀表格不會繼承邊框的樣式、啟用樣式還有表格的變數

也就是說表格裡面有表格，每個表格要分別套用樣式




表尾

在表尾加上 tfoot 會增加表尾的欄位




表格標題

加上 <caption> 可以幫表格加上標題




<table> 的 class 加上 caption-top 就可以把標題放在最上面




響應式表格

隨視窗大小響應式

可以在 <table> 外層的 div 的 class 加上 table-responsive 讓表格變成響應式表格

表格會隨著視窗大小變化，比較長的表格可以水平滾動




特定斷點

也可以直接指定斷點

在 table-responsive 後面加上 -sm、-md、-lg、-xl、-xxl

像是 table-responsive-lg

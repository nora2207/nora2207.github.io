資料雙向綁定的 v-model
Nora
  2024-10-30 16:12:00   2024-11-20 20:11:39  
  • Vue
  

input

最基本的雙向綁定

用 input 來綁定 v-model

資料會跟著 input 裡面的內容改變



textarea

綁定到 textarea 上

和前面的 input 結果一樣



checkbox

單選框




第一個是單選框的部分

v-model 綁定 checkAnswer


一開始的 checkAnswer 為 false


然後利用三元運算子來分別顯示吃飽了和還沒

當布林值為 false 的時候會顯示 還沒

true 的時候則會顯示 吃飽了


單選延伸

前面是利用三元運算子來呈現

這邊直接回傳單一的值來顯示


v-model 綁定 checkAnswer2

然後設定 true-value 和 false-value 分別為吃飽了和還沒

這樣一開始不會顯示內容

但點了 checkbox 後就會顯示吃飽了或是還沒


複選框

定義 checkAnswer3 為一個 陣列


然後把選項分別綁定 v-model

這樣勾選時就會把那個 value 值加到陣列裡面去


radio 單選框

和剛剛前面的 checkbox 有點類似

不過 radio 是單選，而 checkbox 是複選

所以勾選時只能選一個選項

換選項時候把剛剛的選項給覆蓋過去


一開始定義時，直接給一個預設值


接著把選項分別綁定 v-model

勾選時就會出現目前選的選項了



select

單選

先定義一個空字串


使用 v-for 和 key 將資料渲染出來

選項裡面顯示的是名稱和金額

使用 :value 可以動態綁定 (如果是 value 則為固定的值)

如果沒有 value 的話會代入選項的整段文字


value="" 預設會顯示 說吧，你要吃什麼？

disabled 可以讓之後不能選 說吧，你要吃什麼？

只能選其他的選項



多選

多選的部分則是定義一個空陣列


其他和單選的部分一樣

使用 v-for、v-model、v-bind

不一樣的地方是加入 multiple

這樣就可以變成多選了 (需要搭配鍵盤的 ctrl)

Vue 起手式

一開始先在 HTML 中定義一個 dom 元素 app

id 的名稱可以自己修改


接著要來定義 Vue 應用程式

這個部分要寫在 JS 中

首先先定義一個變數指向一個物件

然後透過 Vue 的 createApp 方法

把應用程式生成到畫面上的 #app 位置


如果是放在 HTML 中

要定義一個 script 區塊

而且 script 要放在 dom 元素下面才能讀取到


Vue 元件

Vue 元件有資料、事件觸發、生命週期

data 資料

資料一定是函式

會回傳一個物件，就是我們定義的資料內容


要在裡面加資料，就在 return 裡新增變數

接著要新增到畫面上

就用兩個大括號的形式加上變數名稱

這樣就可以把剛剛值 0 反應到畫面上了


created 生命週期

和資料一樣也是函式

生命週期是元件生成的必經過程

如果有些資料是要在進入元件後才調整

就可以在生命週期裡面進行調整

像是一開始建立元件時還沒有資料

但是進入元件後會從遠端取得資料

就會使用到生命週期的方法


這裡把值改成 10

因此畫面上的數字就從 0 變成 10 了


methods 方法

和前兩個不同，方法是物件

可以用各種方式來觸發


這裡新增一個按鈕，並且在點擊時數字會改變


在 methods 新增一個 clickMe() 函式的方法


要觸發 methods 裡的方法時

可以使用 v-on 來觸發按鈕


接著讓每次點按鈕時，counter 的值都會 +1

這樣就成功了


最後是預覽畫面

用 v-bind 綁定 HTML 屬性
Nora
  2024-10-29 15:52:00   2024-11-19 18:13:05  
  • Vue
  

要把資料內容綁定到 HTML 屬性上，可以使用 v-bind

v-bind 有好幾種屬性

像是圖片 src、驗證 disabled 之類的


v-bind 也可以縮寫

如果原本為 v-bind:src，縮寫直接去掉前面的 v-bind

變成吂號加上屬性名稱像是 :src 就可以了

搭配 v-for

key

搭配 v-for 來使用

也可以使用 v-bind 來綁定 key 值


ID

使用 v-bind 為每個 input 綁定獨立的 ID

這裡使用 name 來當作 ID

input 的 :id 和 label 的 :for 搭配在一起

這樣就可以在點選 checkbox 時選到對的選項

使用動態切換的方式來改變圖片尺寸

一開始的圖片大小設定為 200


接著設定手動改變圖片的大小

範圍是 150 到 800

移動橫桿就可以改變數值


前面是圖片的網址

後面圖片的大小用的是動態的變數



綁定動態屬性

宣告 dynamic 這個變數

然後使用三元運算子的方式

如果是 disabled 就會切換成 readonly

如果不是 disabled 就會切換成 disabled

也就是會在 disabled 和 readonly 之間做切換


接著要讓 input 在 readonly 時可以選但無法輸入文字

在 disabled 時則完全無法選的狀態

可以使用中括號裡面加變數 :[dynamic] 來綁定動態的值



class

v-bind 加上 class 有兩種方式

一種是物件，一種是陣列

這裡先介紹物件的方式


物件寫法

物件的 key 值是對應 className

這裡的 css 的樣式名稱為 rotate

而物件的值則是判斷式

也就是{ key 值: 判斷式 }


如果 className 的名稱裡面有 -

就需要框起來用字串的形式呈現才可以


如果兩個要同時呈現，就在中間加逗號就可以了


接著如果要點按鈕時切換樣式

可以先設定這兩個判斷式

設定按鈕點擊時切換 true 和 false


再替換原本的判斷式部分就可以了

style

值則是帶入 style 相對應的值

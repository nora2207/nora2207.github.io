position 的設定值

absolute 絕對定位

產生在父元素中的絕對位置


依照設定的位置顯示

全部一樣設定，所以多個 div 會重疊在一起

(設定透明度可以看出重疊的部分)




relative 相對定位

產生相對父元素的位置


前面怎麼排，後面就跟著排




fixed 固定在網頁上

固定在網頁的某個位置

也就是說 (使用 position: fixed; 就可以把元素固定在畫面上)


下面這個例子在 div 有設定 top 和 left

但是 div 沒有設定 position，所以這裡的設定是 static

另外在 four 這個 div 有設定 fixed

所以可以依 top 和 left 固定位置

這個時候就算捲動捲軸

這個 div 還是一直都在原本的位置，不會隨著頁面移動




再舉另外一個例子

這裡設計了一個橘色的方框框

利用 bottom: 0; 和 right: 0; 就可以固定在右下角

滑鼠在滑動內容時，也不會被影響，會一直釘在右下角




static 預設值

top、bottom、left、right、z-index的設定都沒有作用

會依瀏覽器預設自動排版位置




inherit 繼承父元素的設定值

設定這個值，會繼承父元素的 position 設定

網路請求的種類
Nora
  2024-01-28 20:39:29 2024-01-28 20:39   2024-04-24 21:38:11  
  • JavaScript
  

HTTP 請求方法比較常用的有下面幾種

GET

使用 GET 的方式來請求取得資料

像是輸入網址後瀏覽器會顯示相關的資訊


POST

把資料傳送給對方

像是傳送帳號密碼到伺服器看看有沒有吻合的

有就回傳正確，沒有就回傳錯誤


請求資料格式 request headers 中的 Content-Type 可以告知對方我們要傳送的格式

比較常見的有下面幾種

application/x-www-form-urlencoded

像是 form 表單預設按下提交按鈕時會傳送到指定的網址去


application/json

這種是透過 JSON 格式來傳遞參數資料

像是用 axios 套件發送網路請求時，預設也是這種格式


multipart/form-data

要傳送檔案(像是圖片、PDF 或是 MP4…之類的)格式時

Content-Type 就是要用這一種


text/plain

記事本格式，比較不常用


DELETE

刪除資料

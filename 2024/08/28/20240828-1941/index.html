設計簡易型的計數按鈕
Nora
  2024-08-28 19:41:00   2024-09-08 20:07:23  
  • JavaScript
  

按下按鈕時，可以加一或減一




先設計計數器和加減按鈕


count 一開始是 0

因為每次變動要減 1，所以用 count--

接著把數字放到計數器的欄位


然後在 button 的地方放 onclick="minus()"

這樣每次點按鈕時就會執行 minus()

數字就會以每次減1的方式減少了


加法的部分也是一樣的做法

只是 count-- 改成 count++

變成每次都加 1


這裡的用了兩種方法把內容放到網頁上
textContent 單純放文字
innerHTML 可以把 HTML 標籤放上去

CSS動畫效果-過渡 transition
Nora
  2024-08-02 20:34:00   2024-09-21 17:53:27  
  • CSS
  

特性

transition 有四個特性可以設定，分別是

transition-property 進行過渡的 CSS 特性名稱

transition-duration 過渡時間

transition-timing-function 速度曲線函數

transition-delay 延遲時間


也可以直接使用 transition 來設定上面這四個項目


可以設定的特性

顏色 color

可以設定顏色的類別：

  • 背景色 background-color

  • 邊框色 border-color

  • 文字顏色 color

  • 輪廓色 outline-color


長度 length 或 百分比 percentage

除了數值，也可以用百分比表示

  • 邊框寬度 border-width

  • 外邊距 margin

  • 內邊距 padding

  • 高度 height

  • 上 top

  • 下 bottom

  • 左 left

  • 右 right

  • 背景位置 background-position

  • 字大小 font-size

  • 文字間距 letter-spacing

  • 單字間距 word-spacing


數字

  • zoom 縮放

  • font-weight 字型粗細

  • 透明度 opacity


整數 integer

  • z-index Z軸堆疊順序

  • outline-offset 輪廓與邊框距離


其他

  • text-shadow 文字陰影

  • background-image 背景圖

  • visibility 可見度

  • line-height 行高

  • vertical-align 垂直對齊


速度曲線

過渡效果的速度曲線，有下面幾種

  • linear
    相同速度 =cubic-bezier(0,0,1,1)

  • ease
    慢速開始、變快、慢速結束=cubic-bezier(0.25,0.1,0.25,1)

  • ease-in
    慢速開始=cubic-bezier(0.42,0,1,1)

  • ease-out
    慢速結束=cubic-bezier(0,0,0.58,1)

  • ease-in-out
    慢速開始、結束=cubic-bezier(0.42,0,0.58,1)

  • cubic-bezier(0,0,0,0)
    自訂義數值

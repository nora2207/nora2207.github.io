上面設計一個按鈕

按下按鈕可以隨機產生 1~49 之間的任一個數字

下面則是開獎號碼的欄位


隨機產生數字

開獎號碼的部分設計了六個

是將 input 放在 form 裡面

這六個 input 的 name 分別是 n1 ~ n6


接著要定義這六個數字

先使用 Math.random() 可以取得 0~1 之間的隨機小數

因為我們需要的是 1~49

所以先乘 49 會得到 0~49 之間的隨機小數

因為我們需要整數，所以用 Math.floor() 無條件捨去

這時會得到 0~48 之間的整數

最後要再加 1，變成 1~49 才會是我們要的數字


Math.floor() 為無條件捨去後的值
Math.round() 為四捨五入後的值


接著在點按鈕時，執行 function 就會產生號碼了




數字不重複

用上面的方法可以分別隨機取得六個數字

但是每一個數字都是獨立的，所以數字有可能會重複

像是 31 14 5 31 26 42 中的 31 出現了二次

如果想要每次出現的一組六個數字都不要重複的話

可以用下面的方法修改 JS 的程式碼


使用 if 來檢查數字

先定義一個空陣列


產生一個範圍在 0 ~ max 之間的隨機整數數字


接著用 if 來檢查數字是否有重複

檢查數字 num 有沒有在 numbers 這個陣列裡面

如果數字不在陣列裡，就用 numbers.push(num) 把數字加進去

如果數字已經存在了，就不做任何動作


就這樣一直循環直到數字達到需要的數量 count


接著把需要的數量和範圍代進去

需要 6 個數字，最大的數字是 49

這樣就可以產生六個不重複的數字了



用 Set() 來處理

使用 Set() 可以讓陣列裡面的元素不會重複

這樣可以過濾掉陣列中重複的元素，只留下不一樣的


建立 numbers 這個 set


產生一個範圍在 1~49 之間的隨機整數數字

直到 set 中有 6 個數字

(每次的數字在 numbers 中沒有重複，就放進 numbers 中)


使用 Array.from() 將 set 轉換為陣列

接著就可以將陣列的內容放到六個 input 裡面了

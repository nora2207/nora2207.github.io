用 vue 來製作飲料點餐的工具-上
Nora
  2024-11-10 20:33:00   2024-12-08 19:56:04  
  • Vue
  

基礎版本的飲料點餐工具，分成左、右上、右下三個區塊

左邊是飲料的品項

選擇品項之後，右上區塊可以選擇數量、冰塊和甜度

然後可以加到清單(右下)中，同時也會計算金額




首先先來看一下原始框架的部分

這個是左邊的飲料品項欄位

(這邊都先把 class 的部分刪掉，比較簡潔)


這裡是資料的部分

還有所有的飲料品項


接著來要開始把指令套進去

左方飲料品項

先用 v-for 渲染資料，記得要搭配 key


然後根據項目名稱 name、engName、price

用雙大括號把資料渲染到畫面上

這樣左邊的飲料品項就出現了


準備暫存資料

因為vue是關注點分離，所以需要先定義一個暫存的資料內容


然後在 methods 定義一個方法


設定名稱時
變數則是名詞寫在前面，像是 itemSelected
方法會把動詞寫在前面，像是 selectedProduct


接著在 a 標籤加入 @click 方法，記得加上 prevent 避免原生的 html 事件影響

這裡的 selectedProduct(item) 中的 item 和 selectedProduct(product) 中的 product 是一樣的東西


然後把資料放進去方法裡

用 ...product 結構的形式避免影響原始資料


接著把樣式套回 a 標籤中，並且判斷選擇的品項和迴圈中的品項相同時，會套上 active 的樣式


這樣在點飲料品項時，這個飲料 item 會傳到 selectedProduct(product) 的 product 中，然後再暫存於 itemSelected 裡面


右方請先選擇飲品

右方選擇區域一開始是沒有辦法選的

必須要在左邊先選擇飲料後，右邊才可以選甜度、冰塊等

所以這裡加上 v-if 判斷式

選擇飲品後，這個半透明黑色的部分才會不見


暫存資料轉為訂單內容

先定義一個購物車列表，一個空的陣列


新增一個可以加品項進去的方法

這裡的參數 product 就是 itemSelected

(可以重新自訂一個名稱)


在加入按鈕加上 @click


使用 push 在點加入時會將品項加到購物車中


然後是右邊下方的清單表格區

使用 v-for 渲染購物車 orderList 的資料

因為沒有適合的 key 值，所以直接使用 key 來作為 key 值，但因為是要唯一值，所以前面再加上 order


因為 vue 是由資料驅動畫面，所以可以先加入預設值

這裡加入數量 1


接著把目前的項目(品項、數量)代入進去


計算金額

金額小計的部分，在 addToOrder 要重新整理訂單的內容

所以定義一個當前的(這一杯)飲料訂單內容 order

這裡也是用 ...product 結構的形式把當前的資料拿出來

金額小計 total 的部分為飲料品項價格 * 數量

然後再把自訂的訂單內容 order 給 push 進去


然後是訂單的總價部分

定義一個訂單總金額變數，預設是 0 元

把飲料的品項加進去時會計算總價


接著定義計算總金額的方法


每次計算時先把 orderTotal 清空來算

接著用 forEach 箭頭函式把訂單內容一個一個帶出來

訂單總價格 = 每一個小計相加


然後再加進去 addToOrder 中


客製化選項

首先是數量的部分

v-model 綁定選擇的飲料數量


取消、加入按鈕點擊後需要清空表單的欄位內容

新增一個清空欄位的方法


把 resetOrder 加進 addToOrder 中

每次增加品項時，會把該筆的 itemSelected 內容清空


也把 resetOrder 加到取消按鈕中


然後是加料的部分

用 v-for 把加料的選項帶出來，搭配 key 值

key 值則是用字串加上 key 去產生唯一值


接著就可以用雙大括號把其他的名稱替換過去

input 的 id 和 label 的 for 相同就可以綁定同一個選項，這裡就使用剛剛的 key 值


然後要和資料做綁定，所以新增一個 toppings 陣列，選擇的加料都會放到這個陣列裡面


每個加料選項的 value 會對應到 topping

v-model 綁定選擇的加料


下方的清單表格也加入加料選項

但是加入的內容會是陣列，所以要轉成字串


加料的價格要需要修改一下

這裡假設每個加料品項是加10元

所以 total 就會變成飲料價格加上飲料的加料數量(length) 乘以10之後再乘以飲料杯數


單價的部分為飲料價格加上加料的價格


再來是冰塊和甜度的部分

和剛剛的加料差不多

也是一樣用 v-for 帶入冰塊的選項，

key 值是前面 ice 加上 key 去產生唯一值


id 和 for 一樣也是用剛剛的 key 值

然後設定 value 對應到 ice

也綁定 v-model


預設冰塊為正常冰，甜度為全糖


最後一樣是使用雙大括號把內容顯示在畫面上


然後是備註的部分，新增一個 notice


v-model 綁定 itemSelected 的 notice


如果備註不是空白的，就顯示這個欄位

然後也是用雙大括號的方式顯示

這樣就完成了

用 vue 來製作飲料點餐的工具-下
Nora
  2024-11-12 18:10:00   2024-12-08 23:18:32  
  • Vue
  

接下來增加一些比較進階的功能

飲料部分有一些特殊條件

像是薑母茶只有熱飲

珍珠鮮奶茶因為已經有珍珠了，所以加料不能再選珍珠

然後也新增了下方的訂單列表部分




固定甜度冰塊

以薑母茶來說，冰塊部分只有熱可以選


所以在 html 這裡用 :disabled 來把條件值綁定到 disabled 屬性

如果條件為 true 的時候就會被禁用

意思也就是說

如果 itemSelected 沒有 defaults 這個屬性的話就會是 true，就會禁用

或是 itemSelected 的 defaults 裡的 ice 沒有內容，而且也不等於 ice 的話就會是 true，就會禁用


另外 methods 的 selectedProduct(product) 裡面也增加冰塊的三元運算子條件

如果 product 的 defaults 中的 ice 不是空字串，那就會是裡面的內容，如果是空的那就會預設選在正常冰


另外甜度的部分也是一樣的方法

像是冬瓜茶的甜度是固定的，只能全糖

也是用 :disabled 的方法來禁用其他冰塊選項

接著也是用三元運算子來設定預設的甜度選項


預設加料選項

珍珠鮮奶茶中已經有珍珠了，所以加料就不能再選珍珠，必須把珍珠這個選項禁用


這裡也是用 :disabled 的方法

當條件為 true 的時候，該元素會被禁用（disabled）

當條件為 false 的時候，元素就不會被禁用

所以如果 itemSelected 沒有 defaults 這個屬性的話就會是 true，就會禁用

或是 itemSelected 的 defaults 裡的 toppings 陣列有 topping 項目，就會是 true，就會禁用


所以珍珠鮮奶茶的 toppings 裡有珍珠

因此珍珠這個選項就會禁用，只有其他選項可以選


訂單列表

飲料都加入購物車後，就可以點產生訂單的按鈕

下方就會產生這些飲料的這張訂單


先在 data 裡面新增 checkedOrder 部分

這幾個分別是訂單時間、金額、選擇的飲料、餐點數


用 new Date() 記錄當前的日期、時間，記得轉成字串

餐點數 orderCount 則是用 reduce 來計算，從 0 開始累加每一個飲料品項的杯數

然後上面的把 orderList 清空

也把飲料選擇的客製化欄位清空


接著把資料放到畫面中就可以了

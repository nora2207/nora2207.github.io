在專案中使用 Bootstrap

Bootstrap 是一個開源的前端框架，有 HTML、CSS 和 JavaScript 元件

有方便使用的網格系統、響應式設計還有 JS 插件

可以幫助我們快速的開發網頁

下面這個是 Bootstrap 的官方網站以及六角學院翻譯的中文版

Bootstrap 的英文官方網站

六角學院翻譯的繁體中文版

下面來介紹在專案要使用 Bootstrap 5 的幾個方法


用 CDN 連結載入 是比較簡單可以直接使用的方法

但是如果要使用 SASS 的話，就要用 下載原始碼 或是 NPM 的方式


CDN 連結載入

這個是最簡單、直接的方式

首先先到 Bootstrap 的官方網站首頁

往下滑一點可以看到 Include via CDN

然後直接複製 CSS link 和 JS script 貼到 html 中

像是下面這樣

這樣就可以很簡單快速的直接載入 Bootstrap

然後就可以開始使用了


下載原始碼

先進到 Bootstrap 的官方網站

選上方的「Docs」然後再找到「Getting started」裡面的「Download」

接著找到頁面裡的 Source files 來下載 Bootstrap 原始碼

(要注意是要下載 SCSS ，不要下載到 CSS 的檔案)


使用 SASS 客製化

使用 SASS 客製化流程如下

1.在專案資料夾裡面新增 scss 資料夾 及 index.html

2.在 scss 資料夾 裡面新增 all.scss 和 _index.scss

3.在 all.scss 內容新增 @import“index”

4.剛剛下載的 Boorstrap 原始碼檔案裡面找到 SCSS 資料夾，修改為 boorstrap 後放到專案中的 scss資料夾 裡

5.index.html 內容增加 link all.css 檔案

6.all.scss 內容新增 @import“boorstrap/boorstrap”

7.修改 _variable.scss (在boorstrap資料夾內)

比較常修改的項目(從上到下順序)

  • 色系，像是修改主色、輔助色(大概在81行 $theme-colors)，例如主色修改 ”primary”: #FFDF65,，新的顏色可以直接新增在後面，例如darkGray、lightGray、gray-100、gray-700(通常是用100~900)
  • Option(預設樣式，圓角.陰影)
  • Spacing(間隔，也可以新增)
  • Body(背景文字色)
  • Link(連結)
  • breakpoints斷點

8.修改按鈕文字顏色可以新增客製化SCSS

9.載入需要使用的 Google font


NPM 安裝

先到 Bootstrap 的官方網站首頁

複製 npm i bootstrap(後面會有版本數字)

接著有兩個方法可以安裝


VS Code

在 VS Code 開啟專案的資料夾

然後上方「終端機」選擇「新增終端」

接著在下面的終端機輸入 npm init，然後幾個項目都按 Enter

好了之後貼上剛剛複製的 npm i bootstrap 就會安裝 Bootstrap 了

完成之後就可以開始使用 Bootstrap 了


命令提示字元

另一個方法是叫出電腦的終端機來安裝

以 Windows 系統的部分來介紹

先點「開始」，搜尋的地方輸入「命令提示字元」

接著在終端機輸入 cd 後空一格

然後把專案的資料夾路徑拖過去之後按 Enter

路徑就會變成專案的位置了

接著貼上剛剛複製的 npm i bootstrap 就會安裝 Bootstrap 了

完成之後就可以開始使用 Bootstrap 了


使用 SASS 客製化

使用 SASS 客製化流程如下

1.在專案資料夾裡面新增 stylesheets 資料夾及 index.html

2.在 stylesheets 資料夾 裡面新增 all.scss 和 _index.scss

3.進到 Bootstrap 的官網選上方的「Docs」然後再找到「Customize」裡面的「Sass」，複製下面三個項目貼到 all.scss

4.index.html 內容增加 link all.css 檔案

(如果是 VS Code 記得要點 Watch Sass 來編譯檔案)

5.在 stylesheets 資料夾 裡面新增 helpers 資料夾

6.依照這個路徑 node_modules/bootstrap/scss/variables 找到 variables 和 variables-dark 複製到 helpers 資料夾

7.把剛剛 all.scss 裡面的 @import "../node_modules/bootstrap/scss/variables"; 路徑修改為在 helpers 資料夾 中的 variables

(接下來要修改變數都是在 helpers 資料夾 的 variables 中修改)

8.載入需要使用的 Google font


流程大概是這樣

如果覺得直接載入全部檔案太大或是有些元件不會用到

也可以把 all.css 檔案中的下面這段刪掉，改成載入需要的東西就好

步驟如下

後面有標注的是需要載入的，其他則依自己的需要載入

路徑要記得修改為專案檔案的路徑

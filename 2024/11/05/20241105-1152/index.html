按鍵修飾符

鍵盤事件修飾符，又可以分成下面三種

別名修飾

按下特定的按鍵，但是使用另外一個名稱來稱呼

像這裡的 enter 就是別名，按下 enter 時才會觸發



相應按鍵

如果不想要按下 enter 就觸發

可以在前面加上 shift

就變成要按下 shift + enter 時才會觸發



特定鍵

要按下特定鍵才會觸發

像這個就是要按到 h 這個按鍵才會觸發



滑鼠修飾符

滑鼠事件的修飾符，像是滑鼠的左鍵、中鍵、右鍵

要按下指定的按鍵才會觸發

這裡設定按下滑鼠的右鍵才會觸發



事件修飾符

針對事件的本身來進行修飾

prevent 取消預設

調用 event.preventDefault()

使用 prevent 可以取消預設的事件

可以使用在 a 連結、表單、按鈕等等


stop 防止向外找

調用 event.stopPropagation()

stop 可以防止向外尋找

介紹 stop 前先說一下冒泡事件


冒泡事件

冒泡事件是原生的 html 就有的事件，下面用範例來說明


最外層藍色的是 div

白色的是 DOM 元素 box

最中間的則是 button

這三層有各自的事件

當按下最外層藍色區塊時會觸發 div，顯示 div 此事件被觸發了

但是如果按下中間的按鈕時則三個都會觸發

顯示 button 此事件被觸發了、box 此事件被觸發了、div 此事件被觸發了




要調整冒泡事件可以用 stop

把 stop 加到中間的 box

這樣點擊中間層的 box 時就不會向外尋找

會顯示 box 此事件被觸發了


但是點按鈕時還是會觸發到 box 這層

顯示 button 此事件被觸發了、box 此事件被觸發了



capture 由外而內

添加事件監聽器時使用 capture 模式

capture 把事件改成由外而內


點中間的按鈕時會變成從最外面開始觸發

顯示 div 此事件被觸發了、box 此事件被觸發了、button 此事件被觸發了



self

只有當事件是從監聽器綁定的元素本身觸發時才觸發回調

self 只會觸發自己範圍內的


在三層都加上 self

點到哪層就會觸發自己那一層

點藍色的區塊會顯示 div 此事件被觸發了

點白色 box 會顯示 box 此事件被觸發了

點按鈕會顯示 button 此事件被觸發了



once

once 只會觸發一次

要重新整理才會再次觸發


在三個區塊都加上 once

每個區塊的事件只會觸發一次而已

之後再點也不會再次觸發

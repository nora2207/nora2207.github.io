用 JS 來製作記憶翻牌遊戲
Nora
  2024-11-30 18:15:00   2024-12-16 18:19:52  
  • JavaScript
  

總共有6組12張的動物卡片

兩兩配對成功時，卡片會消失

全部配對完成會跳出訊息恭喜過關




基本結構

先在 HTML 設定一個翻牌遊戲的框架 div

其他的內容都會動態的加到這裡面


接著設定版面的樣式和顏色

卡片的排列用 flex 的方式換行

CSS 的部分用 SASS 的 SCSS 來寫


最外層是 game，裡面是 grid，再裡面有 card

每一張卡片 card 都有正面 front 和背面 back

整個卡片區域用 section (.grid)包起來


卡片資料

卡片正面是灰白色背景的這一面，背面是動物圖案的這一面

設定卡片 card、正面 front 和背面 back 的 div

並把隨機排序後的卡片資料放到 grid 中


設定卡片的寬高、圓角和每張卡片間距等


定義 cardsArray 陣列，有每張卡片的名稱和圖片網址

這裡選了六種動物


每張卡片正面的圖片


一開始先把卡片的背面隱藏起來，這樣等到翻轉時才會顯示出來


卡片翻轉

監聽卡片的點擊事件

只處理沒有翻開而且不是同一張的卡片

所以已經翻開或已經配對的卡片就不處理


選到的卡片會加上 selected 的 class，會有旋轉180度的翻牌效果


卡片配對

利用展開的方式將卡片資料放進去

然後用 Math.random() 隨機取數，再用 sort() 來排列


選擇兩張卡片時會檢查有沒有配對

firstGuess 是第一次選的卡片，secondGuess 是第二次選的卡片

isProcessing 為 true 可以防止在兩張卡片還未翻回來時再翻下一張


如果沒有配對，會把猜測清空，然後重置猜測的計數

接著移除剛剛選中卡片的 selected 這個 class，卡片再翻轉回正面

isProcessing 為 false 表示可以再翻下一張卡片


卡片配對成功時會加上 match 的 class

然後卡片會消失不見


如果全部卡片都配對了，就會執行 checkGameOver


遊戲結束

全部卡片都配對成功了，會顯示過關及重新開始的視窗


重新開始遊戲

遊戲結束後，可以重新開始遊戲

把裡面的內容清空，重新初始化遊戲

用 classList 操控 class
Nora
  2024-04-28 21:09:00   2024-04-28 22:11:38  
  • JavaScript
  • class
  • getAttribute
  

add() 新增 class

classList.add() 可以新增 class

這裡有二個連結

連結1 是原本的樣子，連結2 新增了 big 這個 class

所以字就變大了




remove() 刪除 class

classList.remove() 可以刪除 class

這裡有二個連結，一起把字變大、顏色變橘色

但是連結2 刪除了 link 這個 class

所以就變回原本預設的樣式了




toggle() 切換 class

如果原本有這個 class 就會移除，如果沒有就會新增

這三個連結中，第三個有 big 這個 class

所以字會比較大

但是現在連結2 和連結3 用了 classList.toggle() 之後

連結2 多了 big 這個 class，字就變大了

而連結3 移除了 big 這個 class，字就變回原本的大小了




contains() 尋找 class

classList.contains() 可以取得元素的 class

getAttribute 也可以取得，但是有點不太一樣

如果設定的條件是判斷某一個 class 是否存在

像是按鈕1 只設定一個 class

這個時候 classList 和 getAttribute 都可以選取到


但是如果設定了2個 class

這時就只有 classList 可以選取到

因為 classList 只要有包含某一個特定樣式就可以了

但是 getAttribute 需要完全符合

CSS動畫 animation
Nora
  2024-09-05 21:31:00   2024-09-21 18:19:47  
  • CSS
  

動畫指元素從一個樣式變化成另一個樣式

而且會自己動，不需要滑鼠移過去或點擊之類的

也不需要使用JS就能產生動畫


特性

  • @keyframes 關鍵影格

用來指定動畫的開始、結束或是中途某個時間點應該呈現的樣式

設定的方式可以用百分比(%) 或是 from(0%) 和 to(100%)


  • animation 一次設定多種動畫(簡短語法)

  • animation-delay 被載入到開始播放動畫之間的等待時間

要延遲多久才開始動畫


  • animation-direction 播放的方向

動畫播完之後

要直接重頭開始重複播放還是反方向的播回去


  • animation-duration 播放一次動畫的時間

  • animation-iteration-count 播放次數

不斷重複播放可設為 infinite


  • animation-name 動畫名稱

把 @keyframes 定義名稱，其他動畫就可以直接套用這個名稱來使用


  • animation-play-state 暫停或繼續播放動畫

  • animation-timing-function 加速曲線設定動畫播放的速度

範例

背景色-使用 from to

原本的背景色是粉紅色

設定關鍵影格 @keyframes 是 from 紅色，然後 to 橘色


@keyframes 的名稱 bgAni 也要放在要動畫的地方

然後設定動畫的時間是3秒

這二行也可以簡寫成


這樣就可以看到效果是點進網頁後

方塊會用3秒的時間，由紅色變成橘色

結束動畫後變回原本設定的粉紅色




背景色-使用 %

使用 % 直接指定要在百分之多少的時候改變

這裡設定了每20%換一個顏色

依序是 紅 橘 黃 綠 藍 紫




變色&移動

設定一邊改變背景顏色，一邊移動


因為需要改變位置

所以也要記得加上 position: relative

如果沒加就不會動了




變色&移動-反方向

這裡設定了反方向和3次

因此跑完第1次動畫之後，第2次會是反方向跑回去

跑完第3次動畫之後，第4次會是反方向跑回去




改變邊框-延遲&重複

改變邊框的顏色和粗細


這裡加上了延遲2秒

所以會先等2秒後才開始動畫


也加了重複的次數

動畫會重複3次才停止


如果想要一直重複都不停止

就改成 infinite




改變邊框-縮放

改變邊框的縮放

寬度從原本的 2px 變成 20px，再變回 2px

搭配不一樣的邊框會有不一樣的效果




改變邊框-圓角縮放

改變邊框的四個角進行縮放

從直角變成 60px 的角度




內邊框縮放

增加減少內邊框 padding 的動畫




文字間距

Merry Christmas！ 是用 letter-spacing

所以會以字母為單位移動


而 happy new year！ 是用 word-spacing

所以會以單字為單位移動




文字行距

利用動畫改變文字的行距

原本是 16px 變成 50px




暫停&繼續

設定方塊的移動路徑


在JS分別設定 animation-play-state 繼續和暫停


按下按鈕就會執行 run() 或 stop() 函式

就可以移動或暫停動畫了




角落往中間移動

藍色的鳥原本在左下，星星原本在左上


設計了5秒重複的動畫

在 50% 的時間時，圖片的位置會跑到中間

(藍色鳥的圖片不重複，看的比較清楚)




圖片縮放

花原本的大小是 40px，狗原本的大小是 200px


設計了5秒重複的動畫

圖片的尺寸會變成 100px

因此原本比較小的花會從 40px 變成 100px

原本比較大的狗會從 200px 變成 100px




速度曲線

不同速度曲線函數的動畫

  • linear 固定速度

  • ease 慢開始，中間快，慢結束

  • ease-in 慢開始

  • ease-out 慢結束

  • ease-in-out 慢開始，慢結束

總時間是一樣的，但是移動的速度不太一樣

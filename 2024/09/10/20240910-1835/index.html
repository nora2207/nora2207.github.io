CSS 的選擇器
Nora
  2024-09-10 18:35:00   2024-09-21 19:04:38  
  • CSS
  

元素名稱

HTML 的元素名稱直接設定

像是 div p h1 之類的


一次選取好幾個 HTML 的元素名稱套用相同的樣式

像是 div 和 h1 一起套用背景變成粉紅色


選取 id 來套用樣式


選取 class 來套用樣式


選取特定元素中指定的 class

在所有的 h1 中，class 為 box 的文字改成粉紅色


選取所有的元素

所有元素的字都改成粉紅色

如果是下面這樣就變成 h1 中所有子元素的字都改成粉紅色


元素和元素間

元素內的元素

h1 內的所有 p 修改文字為粉紅色

只要 p 在 h1 內(不管幾層)，就會套用到樣式

如果 p 沒有在 h1 內，則不會套用樣式


h1 ~ p

有 h1 放在前面的所有 p

(h1 和 p 有相同的父元素)

(p 不一定要緊接在 h1 之後)


h1 > p

h1 下一層的所有 p 文字改成粉紅色

(如果 p 在 h1 內，但 h1 不是 p 的父元素，則不會套用)


h1 + p

緊接在 h1 之後(之內不算)的 p 文字改成粉紅色


屬性選擇器

[attribute] 屬性

選取特定屬性的元素

具 target 屬性的 a 元素文字改成粉紅色

如果是下面這樣就變成具 id 屬性的元素文字改成粉紅色


[attribute = value] 精準特定值

選取具有特定屬性及屬性值的元素

target 屬性值為 _blank 的 a 元素文字改成粉紅色

像下面這個就是指 title 屬性值為 pic 的元素背景改成粉紅色

(title 為當滑鼠停留在上面時，會顯示的文字)


[attribute ~= value] 包含完整單字

title 屬性值包含 pic 單字的元素背景改成粉紅色


[attribute |= value] 開頭

選取具有特定屬性及屬性值以特定文字開始的元素

(單獨一個字或是用-連結)

所有 title 屬性值以 pic 為開始文字的元素背景改成粉紅色

下面這個是指所有 class 屬性值以 box 為開始文字的 p 元素文字改成紅色


[attribute ^= value] 開頭

以 value 為開始文字的元素

(沒有限定或要求要獨立的單字)

對於所有 class 屬性值以 box 為開始文字的 p 元素文字改成紅色


[attribute $= value] 結尾

以 value 為結尾文字的元素

(沒有限定或要求要獨立的單字)

對於所有 class 屬性值以 box 為結尾文字的 p 元素文字改成紅色


[attribute *= value] 包含

包含 value 子字串的元素

(沒有限定或要求要獨立的單字)

選取具有特定屬性及屬性值包含特定子字串的元素

對於所有 class 屬性值包含 box 子字串的 p 元素文字改成紅色


其他

也可以設定 type 或是 alt


偽類

:hover 滑鼠移至元素時

滑鼠移到 p 段落時文字顏色變綠色


:active 元素被點擊時

a 被點擊時(點擊當下)文字顏色變綠色


偽元素

::after 插入內容在指定元素後方

在 h1 後面插入綠色的 hello


::befor 插入內容在指定元素前方

在 h1 前面插入紅色的 hello


狀態

:checked 選項被選取時

只能用在 radio 或 checkbox 被選取後改變樣式

如果只要針對某一種可以這樣寫


:disabled 選項無法作用時


:enabled 選項可以作用時


:empty 元素內沒有內容或子元素時


子節點

:first-child 父元素內第一個子元素的樣式

li 的父元素是 ul，所以設定 ul 下的第一個 li 背景為粉紅色


:last-child 父元素內最後一個子元素的樣式

li 的父元素是 ul，所以設定 ul 下的最後一個 li 背景為粉紅色


:nth-child() 父元素內第 n 個子元素的樣式

設定第二個li子元素的樣式

裡面放不同的數字，可以指定不同的位置

如果要設定第奇數個，可以用 odd

第偶數個則為 even

第4的倍數餘1個則為 4n+1


:nth-of-type() 父元素內指定型別第 n 個子元素的樣式

和 :nth-child() 類似，但是只計算 li


:nth-last-child() 父元素內倒數第 n 個子元素的樣式

和 :nth-child() 相反

這個是指倒數第二個 li


:nth-last-of-type() 父元素內指定型別倒數第 n 個子元素的樣式

和 :nth-of-type() 相反

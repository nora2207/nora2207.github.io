JavaScript ES6 縮寫
Nora
  2024-09-02 20:22:00   2024-10-21 11:30:46  
  • JavaScript
  

縮寫可以讓 JavaScript 語法看起來更簡潔

下面來介紹一些縮寫方法


物件字面值 Object literals

針對原有的大括號形式定義物件的方式


物件內的函式

可以直接把 :function 省略掉

省略後的運行方式也是一樣的

不會影響結果



物件內的變數

前面的名稱和後面的值一樣時

可以省略同名的部分，把前面的名稱刪掉


展開

不同陣列合併

把兩個不同的陣列合併在一起


物件擴展

新增一個物件包含新方法，同時加入原有的方法


轉成純陣列

如果類陣列的方法不足

就可以用展開的形式把陣列轉成純陣列


預設值

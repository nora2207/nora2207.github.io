Bootstrap 的 Flex
Nora
  2025-01-10 18:30:00   2025-01-17 18:31:50  
  • Bootstrap
  

flex 包含行內和區塊兩種

大部分以區塊為主，行內較少使用

在 class 加上 d-flex(上圖) 或是 d-inline-flex(下圖)



方向性

用 .flex-row (上圖)來設定水平的方向(瀏覽器預設值)

或是用 .flex-row-reverse (下圖)來設定水平方向的反轉




用 .flex-column (上圖)設定垂直方向

或是用 .flex-column-reverse (下圖)來設定垂直方向的反轉




Justify content

  • justify-content-start
  • justify-content-end
  • justify-content-center
  • justify-content-between
  • justify-content-around
  • justify-content-evenly


Align items

  • align-items-start
  • align-items-end
  • align-items-center
  • align-items-baseline (對齊行高)
  • align-items-stretch


Align self

內層元素的對齊方式

  • align-self-start
  • align-self-end
  • align-self-center
  • align-self-baseline
  • align-self-stretch


Auto margins

用 margin auto 來推移區塊



搭配 align-items 使用



換行 Wrap

瀏覽器預設 .flex-nowrap 完全不換行

或是可以用 .flex-wrap 換行，用 .flex-wrap-reverse 反向換行



Order

只能使用從 0 到 5 的整數值，任何額外值則需要自定義的 CSS



Align content

***這個特性對於單行的 Flex 項目沒有作用

  • align-content-start
  • align-content-end
  • align-content-center
  • align-content-around
  • align-content-stretch


Media object

加入 .flex-row-reverse (下圖) 為區塊反轉

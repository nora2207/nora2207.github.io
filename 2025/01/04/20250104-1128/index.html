Bootstrap 排版技巧及常見錯誤
Nora
  2025-01-04 11:28:00   2025-01-05 11:30:35  
  • Bootstrap
  

排版小技巧

置中圖片

不需要所有內容都佔滿 12 欄寬

使用 container > row justify-content-center > col-md-8




基本三欄式

可以用 flex 的特性，將部分內容進行推移達到等高的視覺效果

使用 d-flex、flex-column 和按鈕加上 mt-auto




交錯式排列

只有主要內容需要套用容器、格線，背景可以不需要套用

使用 container > row、mb-4、justify-content-center、align-items-center (中間部分反轉 flex-md-row-reverse)> col-md-5




左選單、右內容

可以善用 gutter 的大小差異，製作不同間距的格線

相同視覺群組，距離會較為接近

在右方群組加上 gx-2




常見錯誤

外層缺少 container

外層沒有 .container

下方會出現卷軸，而且內容也會太貼齊邊邊




row 內層加入 col 以外的內容

在 row 的內層加入 .col 以外的內容




都是 12 欄，可以不需要使用格線功能

col-md-12 和 col-12，刪除後結果一樣



行動版 col-12

.col-md... 在行動版都是 .col-12，因此不需要加入

下方2個 col-12，刪除後結果一樣




調整格線系統的水平間距

調整格線系統的水平間距是有風險的行為

上方左右兩個的寬度變不一樣，所以盡量不要調整 column 裡面原本的 padding

下方兩個對齊的位置不太一樣，行動版會比較明顯，而且還出現水平卷軸

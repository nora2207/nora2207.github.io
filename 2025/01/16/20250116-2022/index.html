Bootstrap Position 位置介紹
Nora
  2025-01-16 20:22:00   2025-01-17 18:37:43  
  • Bootstrap
  

排列元素 Arrange elements

格式為 {property}-{position}

property 為以下的其中一種:

  • top
  • start
  • bottom
  • end

position 為以下的其中一種:

  • 0 - 0
  • 50 - 50%
  • 100 - 100%

元素加上 position-absolute 和 position-relative 改變定位形式

然後再透過 {property}-{position} 的形式加上定位


要使用絕對定位，外層一定要使用相對定位




元素置中 Center elements

可以透過轉換通用類別 .translate-middle 將元素置中

效果就是好像有對齊在什麼線上一樣


加上 .translate-middle-x 或 .translate-middle-y 類別

元素只能在水平或垂直方向上定位


範例：通知

下面有三個按鈕



第一個和第三個是用 top-0 start-100 translate-middle

第二個是用 top-100 start-50 translate-middle


範例：進度條

顯示目前進度到哪邊



在三個數字按鈕上加上 top-0 start-*{ 0 50 100} translate-middle


固定在上方 Fixed top

用 fixed-top 把元素固定在螢幕的頂端




固定在下方 Fixed bottom

用 fixed-top 把元素固定在螢幕的底部




選單固定在上方 Sticky top

滾動卷軸時，左邊的選單可以一直固定在畫面上

如果上面有 Navbar 之類的被擋住畫面可以加上 style="top:80px" 就可以把選單往下移




Sticky top 也可以響應式

  • sticky-sm-top

  • sticky-md-top

  • sticky-lg-top

  • sticky-xl-top

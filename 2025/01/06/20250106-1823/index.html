Bootstrap 顏色變數
Nora
  2025-01-06 18:23:00   2025-02-11 17:35:46  
  • Bootstrap
  

背景 Background color

情境色：

  • primary 藍
  • secondary 灰
  • success 綠
  • info 亮藍
  • warning 黃
  • danger 紅
  • light 白
  • dark 黑

情境色都可以套用在背景色，另外還有下面這幾個

  • bg-body (指 body 的顏色，預設是白色)
  • bg-white
  • bg-transparent (透明色)


漸層色

calss 加上 bg-gradient 可以套用漸層色

這個功能預設不會開啟，需要調整變數才能啟用



文字 color

文字顏色可以使用 .text-{ 情境色彩 } 來套用

不適合用在連結上，連結要用 link 加情境色彩


連結顏色 Colored links

連結顏色可以使用 .link-{ 情境色彩 } 來套用

會有 :hover 和 :focus 的狀態

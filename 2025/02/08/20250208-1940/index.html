基本介紹

  • 加上 .form-control、.form-label、.form-text

  • label for 與 input id 有正確對應

  • input type 使用正確，例如 email 用於電子郵件地址、number 用於數字

  • button 有正確套用 type


像是下面這個表單

在 label 的部分加上 .form-label

在 input 加上 .form-control

input 下方的說明文字加上 .form-text

最下面的按鈕 type="submit" 才可以送出，type="button" 是沒辦法運行的


We'll never share your email with anyone else.



表單尺寸

表單加上 .form-control-{sm,lg} 可以改變不同尺寸




表單禁用和僅能閱讀

表單加上 disabled 屬性

雖然都不能選取，但 readonly 會有選取的外框，disabled 沒有



檔案 input

要用 type=”file”



顏色選擇

input 加上 .form-control-color 樣式

要用 type="color"



範圍

加上 .form-range 可以設定範圍

加上 min 及 max 屬性也可以設定 min 和 max 的數值

也可以加上 step 屬性，變成間距數值，像是 step="10" 就是一次跳10個數字
